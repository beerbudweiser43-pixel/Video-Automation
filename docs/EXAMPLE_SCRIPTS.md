## Customization Tips

### For Your Own Scripts:

**Before generating:**
- Make script 1.5-3 minutes minimum (unless short-form)
- Include one clear hook in first 30 seconds
- Include one clear call-to-action at end
- Keep sentences roughly 10-20 words each

**Template Selection:**
- Match template to your script's theme
- Use template voice recommendation
- Use template color palette

**Style Selection:**
- "Text + VO" = fastest (1-2 hours)
- "Cinematic" = most professional (3-4 hours)
- "Interactive" = highest engagement (4-6 hours)
